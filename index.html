Featured Projects

Core Infrastructure

Mahall

Global ecommerce infrastructure platform allowing businesses to operate custom, multi-tenant online storefronts.

The Problem

Standard marketplace options force vendor lock-in, high fee structures, and limited configuration options. Meanwhile, self-managed setups require complex infrastructure provisioning beyond the capabilities of medium-sized businesses.

The Solution

Mahall provides a robust multi-tenant platform where merchants launch independent online storefronts with unified product listings, store layouts, customer management, manual payment verification workflows, and feature gating.

Architecture

A multi-tenant architecture utilizing subdomain and custom-domain routing, separate tenant configuration stores, R2-backed media management, and PostgreSQL database isolation. Optimized via a Redis layer for tenant settings and page caches.

Technologies

Go PostgreSQL Redis Next.js Docker Cloudflare R2 Storage

Business Impact

Allows retail and distribution companies to deploy native, branded stores globally in minutes, scaling autonomously without individual database provisioning overhead or high transaction platform fees.

Operations SaaS

TARDOC

Automated medical billing SaaS simplifying claim submissions and administrative workflows for healthcare providers.

The Problem

Healthcare providers waste substantial resources on slow, manual billing procedures and error-prone claim filings. Administrative overhead reduces operational efficiency and delays payment processing cycles.

The Solution

TARDOC introduces transactional automation to medical billing, streamlining claims processing and billing tracking through structured data engines and simplified practitioner dashboards.

Architecture

Structured billing validation engines with secure data-handling layers, event logs for transactional audits, and asynchronous job queues to handle external system synchronization securely.

Technologies

Go Python PostgreSQL Redis Docker Cloudflare

Business Impact

Reduces administrative work hours by automating repetitive filing verification loops, directly lowering error rates and decreasing time-to-collection for medical services.
